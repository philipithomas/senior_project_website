--- layout: post title: Old Website Now on Github ---

My old philipithomas.com website is now available on Github as an open-source project, licensed under an Apache License, version 2.0.

I developed this website theme from scratch for philipithomas.com, and used it for about a year. I just recently switched to a wordpress-based website to allow for more dynamic content, and have decided to open-source the old code. So, use the CSS as you wish, and I set up the old site with a settings file that makes adopting it for a personal website straightforward!

The theme is minimalist, and works well for PHP-based projects or static HTML sites.
